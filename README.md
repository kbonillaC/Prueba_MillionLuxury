
# Luxury API

Proyecto para la prueba tecnica de Million

### Caracteristicas:
- Seguridad de token JWT,
- CQRS pattern

## Pre-Requisites

- Git
- NETCORE 8.0
- Visual Code o Visual Studio



## Puesta en marcha

Clonar el proyecto

```bash
  git clone https://github.com/kbonillaC/Prueba_MillionLuxury.git
  cd Prueba_MillionLuxury
```

Iniciar la Api
```bash
 cd Luxury.Api
 dotnet run
```
